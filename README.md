# 50 Projects in 50 Days - HTML, CSS & JavaScript

A collection of 50+ mini frontend projects built with **HTML, CSS, and vanilla JavaScript**.  
This repository is designed for practicing core web development skills, including DOM manipulation, responsive layouts, CSS animations, API integration, UI components, forms, sliders, games, and interactive web apps.

These projects are based on the **50 Projects in 50 Days** course by kaizen12219.

---

## What You Will Practice

This repository is useful for improving practical frontend development skills, including:

- HTML5 structure and semantic markup
- CSS3 styling, layouts, transitions, and animations
- Vanilla JavaScript and DOM manipulation
- Event handling and browser APIs
- API-based frontend projects
- Responsive and interactive UI components
- Mini games and utility applications
- Frontend portfolio project ideas

## Featured Project Categories

### UI Components

Projects such as Expanding Cards, Progress Steps, FAQ Collapse, Toast Notification, Custom Range Slider, and Feedback UI Design help practice reusable frontend interface patterns.

### CSS Animations

Projects such as Rotating Navigation Animation, Blurry Loading, Scroll Animation, Button Ripple Effect, Kinetic Loader, Auto Text Effect, and 3D Background Boxes focus on transitions, transforms, keyframes, and animation timing.

### JavaScript Apps

Projects such as Dad Jokes, Movie App, GitHub Profiles, Notes App, Password Generator, Quiz App, Todo List, and Simple Timer demonstrate practical JavaScript application logic.

### API Projects

Projects such as Movie App, Dad Jokes, GitHub Profiles, Pokedex, and Random Image Feed use external APIs and asynchronous JavaScript.

### Games and Interactive Projects

Projects such as Insect Catch Game, Drawing App, Sound Board, Random Choice Picker, Drag and Drop, and Hoverboard focus on event handling and interactive browser behavior.
